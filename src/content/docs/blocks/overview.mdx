---
title: Blocos
---

Os blocos são as peças usadas para montar páginas no site do INCT Antirracismo.

Na página [Demonstrativo de Blocos](https://www.inctantirracismo.com.br/pt-BR/demonstrativo-de-blocos) é possível ver diferentes usos dos blocos descritos abaixo.

## Call-To-Action Padrão

Esse é o bloco mais versátil. Nele é possível colocar título, subtítulo, chapéu, conteúdo "rich text", botões e imagem. É possível escolher diferentes esquemas de cores e posicionamentos de texto e imagem.

## Editor de Texto

Esse é o bloco mais comum de sistemas de gerenciamento de conteúdo, também conhecido como WYSIWYG: what you see is what you get. É como se criar um Google Doc, escrever um email ou fazer um post no Wordpress.

## Lista de conteúdo

Esse bloco é usado para páginas que listam tipos de conteúdo do site, como Posts, Pesquisadores, Eventos, Projetos de Pesquisa, Organizações e Publicações. É possível selecionar itens específicos de um tipo de conteúdo ou usar uma [query personalizada em JSON](https://payloadcms.com/docs/queries/overview) para fazer uma consulta ao banco de dados.
