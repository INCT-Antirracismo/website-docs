Coleção de termos definidos para uso no cadastro de conteúdos no site. É uma forma de definir, por exemplo, a lista de Tags disponíveis para uma notícia, ou os possíveis vínculos de uma pessoa com uma Organização, ou os pronomes de uma pessoa.
